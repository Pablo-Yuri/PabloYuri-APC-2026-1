# Visão geral das semanas

Este arquivo reúne as semanas presentes neste workspace em um formato simples de expandir. Sempre que uma nova semana for adicionada, basta criar uma nova seção seguindo o mesmo padrão.

## Como expandir

- Use o título `## Semana XX` para cada nova entrada.
- Mantenha o mesmo conjunto de campos para facilitar leitura e comparação.
- Se a semana tiver arquivos, imagens ou exercícios novos, registre o caminho da pasta correspondente.
